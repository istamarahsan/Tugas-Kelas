Broken Modularazation, Smell ini terjadi jika pada data member, field, atau method yang seharusnya dikumpulkan dalam 1 class/abstraksi malah terpisah dan tersebar di abstraksi lain. pada kasus realnya biasa terdapat class yang berisi data saja (smell data class) dan terdapat class yang menggunakan data class tersebut dalam methodnya. Dalam kasus ini dapat dilihat ada data class yaitu DataMahasiswa dan class Mahasiswa menggunakan data dari DataMahasiswa untuk dimanipulasi. Untuk mengfix smell ini kami memindahkan field milik DataMahasiswa ke class Mahasiswa.
